Detailed Description

Stores two 1D indices and a lambda value for a 3D vertex

Lambda is a value between 0 and 1. A lambda value greater than 0.5 implies that the 3D vert lies closer to v2. A lambda value of 0 would imply that the 3D vert lies directly over v1, and a lambda of 1 implies that it lies completely over v2.
