Detailed Description

If a given MeshRenderer is disabled, this gameobject will disable.

This is useful in the situation where we have the static pointing finger for raycasting mode, and we have a feature that relies on being in raycasting mode. This way, when the static finger model is disabled, this object and its scripts/colliders will also disable
