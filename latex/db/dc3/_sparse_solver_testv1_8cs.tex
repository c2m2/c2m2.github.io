\begin{DoxyCompactList}\small\item\em This is the sparse solver class for solving the Hodgkin-\/\+Huxley equations for the propagation of action potentials. Below are the HH equations this is a system of partial differential equations (PDE), the first equation is for the membrane potential (voltage) it is time spatially dependent and the remaining equations are the ODE equations for the state variables n,m,h (these are unitless). The system is non-\/trivial in that the PDE equation on Voltage is coupled to the ODE equations using n,m,h and these ODE equations are also non-\/linear. \end{DoxyCompactList}\end{DoxyCompactItemize}
