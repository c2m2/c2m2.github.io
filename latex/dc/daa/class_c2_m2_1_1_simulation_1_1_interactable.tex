\doxysubsection{Detailed Description}
Interface for interaction scripts 

\mbox{\hyperlink{class_c2_m2_1_1_simulation_1_1_interactable}{Interactable}} provides an interface so that interaction scripts can affect a \mbox{\hyperlink{class_c2_m2_1_1_simulation_1_1_simulation}{Simulation}} without needing to know its type.
